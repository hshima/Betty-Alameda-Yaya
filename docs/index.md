### Install

To install project make step-by-step

```sh
# Download Project
git clone https://github.com/vitorgja/Betty-Alameda-Yaya.git
# Enter on Project
cd ./Betty-Alameda-Yaya
# Configure .env file
cp default.env .env
```

### Run Project
```sh
# Run Prod
npm start # or npm run start
# Run on Nodemon
npm run nodemon
```
